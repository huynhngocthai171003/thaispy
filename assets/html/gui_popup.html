Layouts Cách bố trí giao diện trong GUIZERO

Pop‑ups (Thông báo bật lên)

Cửa sổ bật lên có thể được dùng để ngắt quãng người dùng bằng cách đặt câu hỏi hoặc hiển thị thông tin.

Sử dụng pop‑ups

Các hộp thoại pop‑up có thể được gọi từ đối tượng App hoặc Window, ví dụ:

app.info("info", "this is a guizero app")

Bạn cũng có thể nhập riêng từng hàm pop‑up ở đầu chương trình, ví dụ:

from guizero import info
info("info", "this is a guizero app")

Mục đích

Các hàm này sẽ hiển thị một hộp thoại để trình bày thông điệp hoặc đặt câu hỏi. Các hàm có sẵn bao gồm:

  • warn(title, text) – hộp cảnh báo với biểu tượng cảnh báo
  • info(title, text) – hộp thông tin với biểu tượng thông tin
  • error(title, text) – hộp lỗi với biểu tượng lỗi
  • yesno(title, text) – hộp có hai lựa chọn “Yes” và “No”. Nhấn “Yes” trả về True; nhấn “No” trả về False.
  • question(title, text, initial_value=None) – hộp có trường nhập text để người dùng nhập trả lời. Nhấn “Ok” trả về chuỗi đã nhập; nhấn “Cancel” trả về None.
  • select_file(title="Select file", folder=".", filetypes=[[ "All files", "*.*" ]], save=False, filename="") – hộp thoại file mở, cho người dùng chọn file. Mặc định là nút “Open”; nếu đặt save=True, sẽ chuyển thành “Save as”. Trả về đường dẫn file đã chọn. Tham số filename có thể cung cấp để tự động điền tên file.
  • select_folder(title="Select folder", folder=".") – hộp thoại chọn thư mục. Trả về đường dẫn thư mục được chọn.
  • select_color(color=None) – hộp chọn màu. Có thể đặt color theo định dạng #rrggbb để chọn màu mặc định khi mở. Nhấn “Ok” trả về chuỗi màu như #rrggbb; nhấn “Cancel” trả về None.

Tất cả các hộp thoại pop‑up sử dụng giao diện native của hệ điều hành, nên sẽ trông khác nhau trên các hệ thống.

Ví dụ

Hộp cảnh báo (Warning box)

Ví dụ sau tạo hộp cảnh báo với tiêu đề "Uh oh!" và thông điệp "You are almost out of biscuits!":

from guizero import App
app = App(title="Biscuit monitor")
app.warn("Uh oh!", "You are almost out of biscuits!")
app.display()

Trên Windows, hộp sẽ trông như ví dụ minh họa (có biểu tượng cảnh báo).

Các hộp info và error hoạt động tương tự, chỉ khác biểu tượng.

Hộp Yes/No

Hàm này trả về giá trị boolean:

  • Nhấn Yes: trả về True
  • Nhấn No: trả về False

Ví dụ sử dụng:

from guizero import App
app = App(title="Snowman")
build_a_snowman = app.yesno("A question...", "Do you want to build a snowman?")
if build_a_snowman == True:
    app.info("Snowman", "It doesn't have to be a snowman")
else:
    app.error("Snowman", "Okay bye...")
app.display()

Đầu tiên sẽ hiển thị hộp Yes/No; nếu Yes, xuất hiện hộp thông tin; nếu No, xuất hiện hộp lỗi.

Ví dụ: dùng alert làm callback

Bạn cũng có thể dùng các hàm này trong callback cho widget. Ví dụ với PushButton, nhấn sẽ bật hộp info:

from guizero import App, PushButton, info
app = App()
button = PushButton(app, command=app.info, args=["Info", "You pressed the button"])
app.display()

Trong đó:

  • app là nơi tạo nút
  • app.info là hàm được gọi
  • args là danh sách đối số cho hàm (title, message)
Ví dụ: hỏi có thực sự muốn đóng?

Bạn có thể dùng yesno để xác nhận người dùng muốn đóng ứng dụng hay không:

from guizero import App, Text

def do_this_when_closed():
    if app.yesno("Close", "Do you want to quit?"):
        app.destroy()

app = App()
title = Text(app, text="blank app")
# Khi người dùng đóng cửa sổ, chạy hàm do_this_when_closed()
app.when_closed = do_this_when_closed
app.display()

Nếu chọn “Yes”, app sẽ đóng; nếu “No”, không làm gì cả.

Ví dụ: đặt câu hỏi (question)

Ví dụ dùng question để hỏi tên người dùng khi nhấn nút:

from guizero import App, PushButton, Text

def button_pressed():
    name = app.question("Hello", "What's your name?")
    # Nếu Cancel, trả về None
    if name is not None:
        hello.value = "Hello " + name

app = App()
button = PushButton(app, command=button_pressed, text="Hello")
hello = Text(app)
app.display()
Ví dụ: chọn tên file
from guizero import App, PushButton, Text

def get_file():
    file_name.value = app.select_file()

app = App()
PushButton(app, command=get_file, text="Get file")
file_name = Text(app)
app.display()

Bạn có thể thay đổi bộ lọc file kiểu:

file_name.value = app.select_file(filetypes=[["All files", "*.*"], ["Text documents", "*.txt"]])

Mặc định là nút Open; bạn có thể chuyển thành Save:

file_name.value = app.select_file(save=True)
Ví dụ: chọn thư mục
from guizero import App, PushButton, Text

def get_folder():
    path.value = app.select_folder()

app = App()
PushButton(app, command=get_folder, text="Get path")
path = Text(app)
app.display()

Bạn có thể đặt thư mục mở mặc định:

file_name.value = app.select_file(folder="c:\\users\\lawsie")
