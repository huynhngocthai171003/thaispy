Chương 1: Giới thiệu về Giao Diện Người Dùng (GUI) Cách cài đặt guizero và tạo ứng dụng đầu tiên của bạn

Bạn cần chuẩn bị gì

Để bắt đầu, bạn cần có:

  • Máy tính (có thể là Raspberry Pi, Mac, Windows hoặc Linux)
  • Kết nối Internet để tải phần mềm
  • Cài đặt các phần mềm sau:
    • Python 3 (tải từ python.org – xem Phụ lục A)
    • Trình soạn thảo mã (IDE) như:
      • IDLE (đi kèm khi cài Python 3)
      • Thonny (thonny.org)
      • Mu (codewith.mu)
      • PyCharm (jetbrains.com/pycharm)
    • Thư viện guizero cho Python (từ: lawsie.github.io/guizero)

GUI là gì và tại sao nên dùng nó?

GUI (giao diện đồ họa người dùng, đọc là “gùy”) là cách giúp chương trình Python của bạn trực quan, dễ dùng và hấp dẫn hơn. Thay vì phải dùng dòng lệnh, bạn có thể tạo các thành phần như nút bấm, hộp văn bản, thanh trượt,... để người dùng tương tác dễ dàng hơn.

Trong cuốn sách này, chúng ta sẽ sử dụng thư viện guizero – được thiết kế đặc biệt để giúp người mới học lập trình có thể tạo giao diện GUI một cách dễ dàng, nhanh chóng.

Guizero thực chất là một lớp bọc đơn giản hơn của thư viện GUI gốc của Python – tkinter. Nhờ đó, bạn không cần lo về những đoạn mã phức tạp.

Cài đặt thư viện guizero

Để sử dụng guizero, bạn cần cài thư viện này. Có hai cách:

Cách 1: Cài bằng dòng lệnh (Terminal hoặc Command Prompt)

pip3 install guizero

Cách 2: Tải trực tiếp từ GitHub dưới dạng file ZIP

Bạn có thể truy cập lawsie.github.io/guizero để xem hướng dẫn chi tiết, kể cả khi bạn không có quyền cài đặt phần mềm trên máy tính.

Viết chương trình đầu tiên: “Hello World”

Sau khi cài đặt xong, ta sẽ viết thử một chương trình đầu tiên có tên là “Hello World”.

from guizero import App

app = App(title="Hello world")
app.display()

Chạy chương trình, bạn sẽ thấy một cửa sổ hiện ra với tiêu đề là “Hello world”.

Thêm các thành phần vào giao diện

Những thành phần như nút, hộp văn bản, thanh trượt,… gọi là widget – bạn sẽ thêm chúng vào giữa đoạn tạo App và dòng app.display().

Ví dụ: thêm dòng văn bản chào mừng vào cửa sổ:

from guizero import App, Text

app = App(title="Hello world")
message = Text(app, text="Chào mừng bạn đến với ứng dụng!")
app.display()

Lưu ý:

  • Ta thêm Text vào dòng import
  • Tạo biến message để hiển thị dòng chữ

Giải thích thêm về dòng mã

Khi tạo một widget (như Text), ta cần:

  • Đặt tên cho nó (ở đây là message)
  • Xác định nó thuộc loại gì (Text)
  • Chỉ định nó nằm trong cửa sổ nào (ở đây là app)
  • Nội dung muốn hiển thị ("Chào mừng bạn đến với ứng dụng!")

Ví dụ đầy đủ: 01-helloworld.py

from guizero import App, Text

app = App(title="Hello world")
message = Text(app, text="Welcome to the app")
app.display()
