Chương 3: Đối tượng Text Hiển thị văn bản tĩnh với guizero

Text là gì?

Text là widget dùng để hiển thị văn bản không chỉnh sửa được trong ứng dụng, thường dùng cho tiêu đề, nhãn hoặc hướng dẫn.

Cách tạo Text

from guizero import App, Text
app = App()
text = Text(app, text="Hello World")
app.display()

Tham số khởi tạo

Tên tham số Kiểu dữ liệu Mặc định Bắt buộc Mô tả
masterApp, Window, Box-CóContainer chứa Text
💬 Giải thích: Là đối tượng cha chứa widget Text.
🧪 Ví dụ: 
from guizero import App, Text

app = App()

app.display()

Phương thức

Phương thức Tham số Trả về Mô tả
after(time, command, args=None) time (int), command (function name), args (list of arguments) - Gọi hàm sau thời gian (ms)
💬 Giải thích: Hẹn giờ thực hiện 1 hành động sau thời gian định sẵn.
🧪 Ví dụ: 
from guizero import App, Text

def chao():
    print("Xin chào sau 1 giây")

app = App()
text = Text(app, text="Chờ chút...")
text.after(1000, chao)
app.display()
append(text) text (string) - Thêm nội dung vào cuối
💬 Giải thích: Thêm văn bản mới vào sau nội dung cũ của widget.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Xin chào")
text.append(" thế giới")
app.display()
cancel(command) command (function name) - Hủy cuộc gọi theo lịch
💬 Giải thích: Hủy hàm đã được đặt bởi `after()` hoặc `repeat()`.
🧪 Ví dụ: 
from guizero import App, Text

def chao():
    print("Sẽ không bao giờ in")

app = App()
text = Text(app, text="Bỏ qua hành động")
text.after(1000, chao)
text.cancel(chao)
app.display()
clear() - - Xóa nội dung
💬 Giải thích: Xóa toàn bộ nội dung văn bản trong widget.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Sẽ bị xóa")
text.clear()
app.display()
destroy() - - Hủy widget
💬 Giải thích: Loại bỏ widget khỏi giao diện.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Sẽ biến mất")
text.destroy()
app.display()
disable() - - Vô hiệu hóa widget
💬 Giải thích: Làm cho widget không thể tương tác.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Không tương tác được")
text.disable()
app.display()
enable() - - Kích hoạt lại widget
💬 Giải thích: Cho phép tương tác lại sau khi bị disable.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Bật lại sau vô hiệu")
text.disable()
text.enable()
app.display()
focus() - - Đặt tiêu điểm
💬 Giải thích: Đưa con trỏ chuột hoặc bàn phím vào widget.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Tập trung vào đây")
text.focus()
app.display()
hide() - - Ẩn widget
💬 Giải thích: Tạm thời ẩn khỏi giao diện.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Sẽ bị ẩn ngay")
text.hide()
app.display()
repeat(time, command, args=None) time (int), command (function name), args (list of arguments) - Gọi hàm định kỳ
💬 Giải thích: Gọi hàm liên tục sau mỗi khoảng thời gian.
🧪 Ví dụ: 
from guizero import App, Text

def dong_ho():
    print("tick")

app = App()
text = Text(app, text="Đồng hồ")
text.repeat(1000, dong_ho)
app.display()
resize(width, height) width (int), height (int) - Thay đổi kích thước
💬 Giải thích: Điều chỉnh lại chiều rộng và chiều cao widget.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Đổi kích thước")
text.resize(30, 3)
app.display()
show() - - Hiển thị lại nếu bị ẩn
💬 Giải thích: Hiện lại widget đã bị `hide()` trước đó.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
text = Text(app, text="Sẽ xuất hiện lại")
text.hide()
text.show()
app.display()

Thuộc tính

Tên thuộc tính Kiểu dữ liệu Mô tả
alignstringCanh lề trong container
💬 Giải thích: Thiết lập vị trí widget bên trong container cha, ví dụ: "top", "bottom", "left", "right", "center".
🧪 Ví dụ: 
from guizero import App, Box, Text

app = App()
box = Box(app, layout="grid")
txt = Text(box, text="Canh lề", grid=[0,0])
txt.align = "top"
app.display()
bgcolorMàu nền
💬 Giải thích: Đặt màu nền cho widget, sử dụng tên màu hoặc mã màu.
🧪 Ví dụ: 
from guizero import App, Text

app = App()
txt = Text(app, text="Xin chào với nền màu")
txt.bg = "lightblue"
app.display()
enabledbooleanWidget đang bật hay tắt
💬 Giải thích: Cho phép hoặc vô hiệu hóa tương tác với widget.
🧪 Ví dụ: 
from guizero import App, TextBox

app = App()
txt = TextBox(app)
txt.value = "Không thể sửa"
txt.enabled = False
app.display()
fontstringFont chữ
💬 Giải thích: Đặt kiểu font cho nội dung văn bản.
🧪 Ví dụ:

txt.font = "Courier"

gridListVị trí lưới
💬 Giải thích: Định vị widget trong bố cục dạng lưới theo hàng, cột.
🧪 Ví dụ: 
from guizero import App, Text
