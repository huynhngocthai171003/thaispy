Kiểu dữ liệu trong Python 6 phút đọc

Kiểu dữ liệu là một khái niệm quan trọng trong lập trình, giúp xác định loại dữ liệu mà một biến có thể lưu trữ và các phép toán có thể thực hiện trên dữ liệu đó. Python là một ngôn ngữ lập trình mạnh mẽ và linh hoạt, hỗ trợ nhiều kiểu dữ liệu khác nhau. Bài viết này sẽ hướng dẫn chi tiết về các kiểu dữ liệu trong Python, bao gồm khái niệm, phân loại và cách chuyển đổi giữa các kiểu dữ liệu.

Kiểu dữ liệu là gì?

Kiểu dữ liệu (data type) là một phân loại dữ liệu mô tả đặc điểm và hành vi của giá trị mà biến lưu trữ. Trong Python, tất cả mọi thứ đều là đối tượng và các kiểu dữ liệu là các lớp được dựng sẵn.

Các kiểu dữ liệu trong Python

Python cung cấp nhiều kiểu dữ liệu cơ bản:

  • Số (Numeric): int, float, complex
  • Chuỗi (String): str
  • Dãy (Sequence): list, tuple, range
  • Nhị phân (Binary): bytes, bytearray, memoryview
  • Ánh xạ (Mapping): dict
  • Boolean: bool
  • Tập hợp (Set): set, frozenset
  • Không giá trị: NoneType

1. Kiểu dữ liệu số (Numeric)

Kiểu dữ liệu số trong Python được sử dụng để lưu trữ các giá trị số. Các giá trị số bao gồm số nguyên (integer), số thực (floating-point), và số phức (complex).

  • int: Lưu trữ các số nguyên, không giới hạn độ dài.
  • float: Lưu trữ các số thực, chính xác đến 15 chữ số thập phân.
  • complex: Lưu trữ các số phức, có dạng a + bj với a và b là các số thực và j là đơn vị ảo.
num1 = 5
print(type(num1))  # int

num2 = 2.5
print(type(num2))  # float

num3 = 1 + 2j
print(type(num3))  # complex

2. Kiểu dữ liệu chuỗi (String)

Chuỗi là một dãy các ký tự được đặt trong dấu ngoặc đơn hoặc dấu ngoặc kép. Python hỗ trợ các ký tự Unicode.

name = 'Python'
print(name)

message = "Python for beginners"
print(message)

3. Kiểu dữ liệu dãy (Sequence Data Types)

Dãy là một tập hợp các mục được sắp xếp. Python có ba kiểu dữ liệu dãy chính: list, tuple, và range.

  • list: Danh sách là một dãy các mục có thể thay đổi, được đặt trong dấu ngoặc vuông [].
  • tuple: Tuple là một dãy các mục không thể thay đổi, được đặt trong dấu ngoặc đơn ().
  • range: Range là một dãy các số nguyên, thường được sử dụng trong vòng lặp.
# List
fruits = ["apple", "banana", "cherry"]
print(fruits)

# Tuple
coordinates = (10.0, 20.0)
print(coordinates)

# Range
numbers = range(5)
print(list(numbers))

4. Kiểu dữ liệu nhị phân (Binary Data Types)

Python hỗ trợ các kiểu dữ liệu nhị phân như bytes, bytearray, và memoryview.

  • bytes: Lưu trữ các dãy byte không thể thay đổi.
  • bytearray: Lưu trữ các dãy byte có thể thay đổi.
  • memoryview: Cung cấp một giao diện để truy cập vào bộ nhớ của các đối tượng khác.
# Bytes
data = b"Hello"
print(data)

# Bytearray
data = bytearray(5)
print(data)

# Memoryview
data = memoryview(bytes(5))
print(data)

5. Kiểu ánh xạ (Mapping)

Kiểu dữ liệu ánh xạ trong Python là từ điển (dictionary), lưu trữ các cặp khóa-giá trị.

person = {"name": "John", "age": 30}
print(person)

6. Kiểu boolean

Kiểu dữ liệu boolean trong Python chỉ có hai giá trị: True và False.

is_active = True
print(is_active, 'là kiểu', type(is_active))

7. Kiểu tập hợp (Set)

Tập hợp là một tập hợp các mục duy nhất, không có thứ tự.

student_ids = {112, 114, 116, 118, 115}
print(student_ids)

8. Kiểu NoneType

NoneType là kiểu dữ liệu đặc biệt trong Python, đại diện cho giá trị không có giá trị.

value = None
print(value, 'là kiểu', type(value))

Chuyển đổi kiểu dữ liệu

Chuyển đổi kiểu dữ liệu là quá trình chuyển đổi dữ liệu từ kiểu này sang kiểu khác. Python hỗ trợ hai loại chuyển đổi kiểu dữ liệu: chuyển đổi ngầm định (implicit conversion) và chuyển đổi tường minh (explicit conversion).

Chuyển đổi ngầm định (Implicit Type Conversion)

Chuyển đổi ngầm định là khi Python tự động chuyển đổi một kiểu dữ liệu sang kiểu khác mà không cần sự can thiệp của người dùng.

a_int = 1
b_float = 1.0
c_sum = a_int + b_float

print(c_sum)
print(type(c_sum))

Chuyển đổi tường minh (Explicit Type Conversion)

Chuyển đổi tường minh, còn được gọi là ép kiểu, là khi người dùng tự chuyển đổi kiểu dữ liệu của một đối tượng sang kiểu dữ liệu khác bằng cách sử dụng các hàm chuyển đổi tích hợp sẵn như int(), float(), str(), v.v. 

num_string = '12'
num_integer = 23

print("Kiểu dữ liệu của num_string trước khi ép kiểu:", type(num_string))

# Ép kiểu tường minh
num_string = int(num_string)

print("Kiểu dữ liệu của num_string sau khi ép kiểu:", type(num_string))

num_sum = num_integer + num_string
print("Tổng:", num_sum)
print("Kiểu dữ liệu của num_sum:", type(num_sum))

Các hàm chuyển đổi kiểu phổ biến

Python cung cấp nhiều hàm chuyển đổi kiểu dữ liệu để chuyển đổi giữa các kiểu dữ liệu khác nhau.

  • int(): Chuyển đổi một giá trị sang kiểu số nguyên.
  • float(): Chuyển đổi một giá trị sang kiểu số thực.
  • str(): Chuyển đổi một giá trị sang kiểu chuỗi.
  • bool(): Chuyển đổi một giá trị sang kiểu boolean.
  • list(): Chuyển đổi một giá trị sang kiểu danh sách.
  • tuple(): Chuyển đổi một giá trị sang kiểu tuple.
  • set(): Chuyển đổi một giá trị sang kiểu tập hợp.
  • dict(): Chuyển đổi một giá trị sang kiểu từ điển.
# Chuyển đổi sang số nguyên
x = int("10")
print(x, 'là kiểu', type(x))

# Chuyển đổi sang số thực
y = float("10.5")
print(y, 'là kiểu', type(y))

# Chuyển đổi sang chuỗi
z = str(10)
print(z, 'là kiểu', type(z))

# Chuyển đổi sang boolean
a = bool(1)
print(a, 'là kiểu', type(a))

# Chuyển đổi sang danh sách
b = list((1, 2, 3))
print(b, 'là kiểu', type(b))

# Chuyển đổi sang tuple
c = tuple([1, 2, 3])
print(c, 'là kiểu', type(c))

# Chuyển đổi sang tập hợp
d = set([1, 2, 3])
print(d, 'là kiểu', type(d))

# Chuyển đổi sang từ điển
e = dict([(1, 'one'), (2, 'two')])
print(e, 'là kiểu', type(e))

Bài tập luyện tập

  • 🧠 Bài tập 1: Tạo biến và in ra bằng f-string
    Tạo các biến để lưu thông tin cá nhân (tên, tuổi, chiều cao, tình trạng là sinh viên) và in ra bằng print(f"...").
  • 🔍 Bài tập 2: Kiểm tra kiểu dữ liệu
    Tạo 4 biến với các kiểu int, str, float, bool. In ra kiểu dữ liệu của từng biến bằng type() và print(f"...").
  • ⟳ Bài tập 3: Ép kiểu
    Tạo một biến là chuỗi số "123". Ép kiểu nó sang int, float, bool, rồi in ra kết quả và kiểu dữ liệu mới của từng biến.
  • 🧰 Bài tập 4: Gán nhiều biến một lúc
    Gán 3 giá trị khác nhau (int, float, str) cho 3 biến trong một dòng. Sau đó in ra giá trị từng biến bằng print(f"...").
