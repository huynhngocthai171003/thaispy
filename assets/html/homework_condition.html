🧠 Bài tập: Câu điều kiện if / elif / else trong Python

🟡 Bài 1: Kiểm tra số lớn nhỏ

  • Nhập một số nguyên
  • Nếu > 100 → in “Số lớn hơn 100”
  • Nếu = 100 → in “Bằng 100”
  • Nếu < 100 → in “Nhỏ hơn 100”

🔵 Bài 2: Xếp loại học lực

  • Nhập điểm trung bình (0–10)
  • Dùng if elif else để in ra: “Giỏi”, “Khá”, “Trung bình”, hoặc “Yếu”

🟢 Bài 3: Số chia hết cho 3 và 5?

  • Nhập một số nguyên
  • Nếu chia hết cho 3 và 5 → “FizzBuzz”
  • Nếu chỉ chia hết cho 3 → “Fizz”
  • Nếu chỉ chia hết cho 5 → “Buzz”
  • Ngược lại in chính số đó

🟣 Bài 4: Tính tiền taxi

  • Nhập số km đi được
  • ≤ 1 km → 20.000đ
  • 1–5 km → 13.000đ/km
  • > 5 km → 11.000đ/km
  • In tổng tiền phải trả

🔶 Bài 5: Kiểm tra năm nhuận

  • Nhập một năm
  • Nếu chia hết cho 400 hoặc (chia hết cho 4 nhưng không chia hết cho 100) → “Năm nhuận”
  • Ngược lại → “Không nhuận”

🔷 Bài 6: Máy tính mini

  • Nhập 2 số và 1 phép toán (+, -, *, /)
  • Dùng if elif else để tính và in kết quả

🟤 Bài 7: Kiểm tra nhiệt độ nước

  • Nhập nhiệt độ (°C)
  • Nếu < 0 → “Nước đóng băng”
  • Nếu từ 0–99 → “Nước ở thể lỏng”
  • Nếu ≥ 100 → “Nước sôi”

⚪ Bài 8: Giờ trong ngày

  • Nhập số giờ (0–23)
  • 0–11 → “Buổi sáng”
  • 12–17 → “Buổi chiều”
  • 18–23 → “Buổi tối”

🟥 Bài 9: Cân nặng chuẩn

  • Nhập chiều cao (m) và cân nặng (kg)
  • Tính BMI = cân nặng / (chiều cao²)
  • BMI < 18.5 → “Gầy”
  • 18.5–24.9 → “Bình thường”
  • 25–29.9 → “Thừa cân”
  • ≥ 30 → “Béo phì”

🟩 Bài 10: Đăng nhập

  • Nhập tên người dùng và mật khẩu
  • Nếu username = admin và password = 12345 → “Đăng nhập thành công”
  • Ngược lại → “Sai thông tin”
